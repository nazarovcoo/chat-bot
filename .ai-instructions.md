# Project Instructions for AI Assistants

## Important Deployment Rule
Whenever frontend files (`.html`, `.js`, `.css`) are modified, the AI MUST ALWAYS automatically deploy the changes to production so the user can immediately see them on `createbotaiagent.com` (Firebase Hosting: `chatbot-acd16.web.app`).

**Command for deployment:**
```bash
firebase deploy --only hosting
```

Do not ask the user for permission to deploy frontend changes; assume that if they ask you to change the UI, they want to see it live immediately.
